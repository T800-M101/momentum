# Memoirly — Personal Journal App

A modern full-stack journaling platform built with Angular, NestJS, Prisma, and PostgreSQL.

---

## Overview

Memoirly is a premium personal journaling application designed as a portfolio-quality SaaS project.

The goal is to showcase:

- Modern frontend architecture
- Scalable backend design
- Clean UI/UX
- Offline-first capabilities
- AI-powered experiences

---

## Tech Stack

### Frontend
- Angular
- TailwindCSS
- Angular Signals
- TipTap Editor

### Backend
- NestJS
- Prisma ORM
- PostgreSQL

### Infrastructure
- Vercel
- Railway
- Neon
- Cloudinary

---

## Core Features

- Rich journal editor
- Mood tracking
- Timeline UI
- Full-text search
- AI-powered summaries
- Offline-first support
- Calendar heatmap
- Dark mode UI
